Dominio, DNS y propietarios

Evalúa la seguridad asociada a la infraestructura del dominio y la confianza.

Aspectos analizados:

DNSSEC

Comprueba si el dominio utiliza DNSSEC para proteger las consultas DNS.

DNSSEC añade firmas digitales a las respuestas DNS, permitiendo verificar que la información recibida es auténtica y evitando que un atacante pueda redirigir al usuario hacia direcciones falsas.

DNS públicos de confianza

Verifica si el dominio responde correctamente desde distintos servidores DNS públicos de confianza.

Una respuesta correcta desde diferentes proveedores DNS indica que el dominio está correctamente configurado y accesible desde múltiples redes.

Antigüedad del dominio

Analiza la fecha de creación del dominio.

Aunque no garantiza la seguridad de un sitio web, los dominios con varios años de antigüedad suelen generar una mayor confianza que aquellos creados recientemente.

Listas negras

Comprueba si el dominio o su dirección IP aparecen en listas negras de ciberseguridad.

La presencia en estas listas puede indicar actividad maliciosa, envío de spam o la existencia de incidentes de seguridad asociados al sitio web.
